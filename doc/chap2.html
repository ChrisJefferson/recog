2. Installation of the recogbase-Package

To install this package just extract the package's archive file to the GAP pkg directory.

By default the recogbase package is not automatically loaded by GAP when it is installed. You must load the package with LoadPackage("recogbase"); before its functions become available.

Usually, this package is only usable if you also install the recog containing all the implemented recognition methods. Since the recog then automatically loads the recogbase the only has to load both packages with LoadPackage("recog");.
