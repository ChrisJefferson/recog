1.1 Philosophy

This package is about group recognition. To be written further.

1.2 Overview over this manual

Chapter 2. describes the installation of this package. Chapter 4. describes the generic, recursive procedure used for group recognition throughout this package. At the heart of this procedure is the definition of "FindHomomorphism" methods, which is also described in that chapter. For the choice of the right method for finding a homomorphism (or an isomorphism) we use another generic procedure, the "method selection" which is not to be confused with the GAP method selection. Our own method selection is described in detail in Chapter 3., because it is interesting in its own right and might be useful in other circumstances.
